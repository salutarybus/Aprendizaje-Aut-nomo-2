# Juego El Ahorcado

## Descripción  
El juego El Ahorcado trata de adivinar una palabra aleatoria mediante un sistema de adivinar letras e intentos.  

## Objetivo  
Implementar un programa que integre los conceptos aprendidos en la asignatura, incluyendo diagramas funcionales, arquitectura de software y programación funcional.  

## Funcionalidades  
- Selección aleatoria de palabras  
- Validación de entrada del usuario  
- Seguimiento de letras adivinadas e intentos restantes  

## Requisitos  
- Python 3.x
